# Extras

Optional utilities kept outside the main pipeline.

- data_collection/squad_mapper.py: squad mapping helpers
- data_collection/squads.py: scrape squads from cdm2026.fr
- data_collection/parsing.py: placeholder for parsing utilities
- evaluation/temporal_cv.py: time-series CV helpers
- evaluation/shap_analysis.py: SHAP analysis helpers
- models/calibration.py: generic calibration helper
- simulation/bookmaker_benchmark.py: bookmaker baseline metrics

These scripts are not wired into Makefile targets or the main pipeline.
